# CardsOffers

App that allow create tasks and do job (be freelancer and be task maker).
In this app used MVP architecture and technologies such as:
- Dagger 
- Retrofit (server on pythonanyway)
- Navigation
- RxJava (for retrofit)
- Room
and other.
